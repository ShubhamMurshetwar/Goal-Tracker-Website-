# Goal Tracker Website

This is a simple and responsive goal tracking website where users can add and track their personal goals, mark them as completed, and view progress over time.

The purpose of this project is to build a clean and easy-to-use web interface that helps users stay organized and focused on their objectives.

## Features
- Add new goals
- View current goals
- Mark goals as completed
- Simple and clean user interface

## Technologies Used
- **HTML** for the structure of the web pages  
- **CSS** for styling and layout  
- **JavaScript** for functionality and interaction
